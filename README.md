## 구현하고 싶은 기능
- <strong>전체선택/전체삭제</strong> : 버튼을 누르면 전체 선택되고, 전체 삭제할 수 있다.
- <strong>전체보기/진행중/완료</strong> : 작성한 목록을 count 해서 진행중과 완료중을 나눠서 볼 수 있게 한다.
- <strong>CSS 개선</strong> : 메모장처럼 디자인 바꾸기
<br>
<br>

## 현재 구현한 기능
- 버튼, Enter 키 누르면 할 일 추가
- 삭제 버튼 누르면 할 일 삭제
- checkbox 클릭하면 할 일 완료 표시 (취소선)
